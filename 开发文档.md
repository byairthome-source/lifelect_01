# Lifelect 开发文档

> Lifelect 擦窗机器人 B2B 品牌官网 — Next.js 14 + TypeScript + Tailwind CSS + PostgreSQL

---

## 1. 项目概述

Lifelect 是面向海外批发商、OEM/ODM 合作伙伴及分销商的 B2B 品牌官网，采用 **Novu Dark Immersive** 设计风格。

- **生产地址**：`https://lifelect-01.vercel.app`
- **本地开发**：`http://localhost:3002`（`npm run dev`）
- **后台管理**：`/admin/login`
- **代码仓库**：`https://github.com/byairthome-source/lifelect_01`

| `Post` | 博客文章 |
| `Customer` | 客户（自动从询价创建） |

### Product 接口

```typescript
interface Product {
  id: string;
  name: string;
  slug: string;
  series: string;          // "M Series" | "D Series" | "BO Series"
  subtitle: string;

  description: string;
  coverColor: string;      // Tailwind 渐变
  image: string;           // 主图 URL
  images: string[];        // 多图 URL 数组
  specs: ProductSpec[];
  features: ProductFeature[];
}
```

## 7. 动画系统

### ScrollReveal 组件 — 6 种变体

| 变体 | 效果 | 应用场景 |
|------|------|----------|
| `fade-up` | 向上淡入 | Why Us 卡片 |
| `fade-in` | 纯透明度渐入 | Project Cases |
| `scale-up` | 从小到大淡入 | 分类卡片、认证徽章 |
| `slide-left` | 从右滑入 | Featured 奇数行、OEM 文字 |
| `slide-right` | 从左滑入 | Featured 偶数行、OEM 图片 |
| `clip-reveal` | 从左向右展开 | 预留 |

### 其他动画

| 动画 | 说明 |
|------|------|
| `animate-float` | 轻微上下浮动（6s 循环） |
| `animate-breathe` | 透明度 + 缩放呼吸（5s 循环） |
| `animate-counter-pop` | 数字弹入效果 |
| `section-divider-line` | 橙色渐变线从中间展开 |
| `animate-bounce-wave` | CTA 文字波浪弹跳 |
| `animate-page-in` | 页面淡入 |

## 8. 部署

### 环境变量（Vercel）

| 变量 | 说明 |
|------|------|
| `DATABASE_URL` | Neon PostgreSQL 连接字符串 |
| `NEXTAUTH_SECRET` | JWT 密钥（32+ 字符） |
| `NEXTAUTH_URL` | 生产域名 `https://lifelect-01.vercel.app` |
| `NEXT_PUBLIC_SITE_URL` | 同上 |
| `BLOB_READ_WRITE_TOKEN` | Vercel Blob token（可选，图片上传用） |

### 本地开发

```bash
npm install
npx prisma db push      # 同步数据库
npx prisma db seed      # 导入种子数据
npm run dev             # → http://localhost:3002
```

### 生产部署

代码推送到 GitHub `main` 分支后，Vercel 自动构建部署。
