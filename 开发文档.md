# Lifelect2 开发文档

> Lifelect 擦窗机器人 B2B 官网 Demo — Next.js 14 + TypeScript + Tailwind CSS 3.4

---

## 1. 项目概述

Lifelect2 是 Lifelect 擦窗机器人的 B2B 品牌官网 Demo，采用 **Novu Dark Immersive** 设计风格，面向海外批发商、OEM/ODM 合作伙伴及分销商。

- **在线地址（开发）**：`http://localhost:3002`
- **启动命令**：`npm run dev`
- **构建命令**：`npm run build`

## 5. 核心架构

### 5.1 页面布局

```
<html>
  <body>
    <Header />           ← fixed, z-50, 10px from top
    <main>
      {children}         ← 各页面内容
    </main>
    <Footer />           ← z-10, bg-novu-near-black
  </body>
</html>
```

### 5.2 首页覆盖式滚动

首页采用 **下层上浮覆顶** 交互模式：

1. `HeroBanner` — `position: fixed; z-index: 0; inset: 0;` 固定占满视口
2. 内容容器 — `position: relative; z-index: 10; margin-top: 100vh;` 初始位于视口下方
3. 用户滚动时，白色容器从底部上浮，渐进式覆盖 Hero
4. 容器顶部 50px 圆角 + 阴影，营造覆盖层次感

### 5.3 Header 自适应对比度

- 使用 `usePathname()` 检测首页 → 仅首页启用 Hero 模式
- 非首页始终使用「白色背景模式」
- 滚动监听：`window.scrollY < window.innerHeight - 80` 判断是否在 Hero 区

| 状态 | 触発条件 | Logo | 导航容器 | 导航文字 | CTA |
|------|----------|------|----------|----------|-----|
| Hero 模式 | 首页 + 在 Hero 视区 | 白色 | 磨砂玻璃 + white/20 边框 | white/80 | 黑底白字 |
| 白色背景 | 非首页 or 滚离 Hero | 黑色 | 磨砂玻璃 + gray-200/60 边框 | near-black/60 | 黑底白字 |

## 6. 数据模型

### Product 接口 (`src/data/products.ts`)

```typescript
interface Product {
  id: string;              // 唯一标识 "w10-l"
  name: string;            // 产品名 "Lifelect W10-L"
  slug: string;            // URL slug "lifelect-w10-l"
  series: string;          // 系列 "W10 Series"
  subtitle: string;        // 副标题
  summary: string;         // 简述（卡片用）
  description: string;     // 详细描述（详情页用）
  coverColor: string;      // Tailwind 渐变类 "from-blue-500 to-blue-800"
  specs: ProductSpec[];    // 9 项技术规格
  features: ProductFeature[]; // 3 项核心特性
}
```

## 7. 开发指南

### 7.1 启动开发服务器

```bash
cd lifelect-website2
npm run dev
# → http://localhost:3002
```

### 7.2 构建生产版本

```bash
npm run build   # → .next/
npm start       # → http://localhost:3000（默认端口）
```

### 7.3 添加新页面

1. 在 `src/app/` 下创建目录 + `page.tsx`
2. 页面自动继承 `layout.tsx` 的 Header/Footer

```tsx
// src/app/about/page.tsx
export default function AboutPage() {
  return (
    <section className="pt-32 lg:pt-40 pb-12 bg-bg-light">
      <div className="container-main">
        <h1 className="section-title">About Us</h1>
      </div>
    </section>
  );
}
```

### 7.4 添加新产品

编辑 `src/data/products.ts`，追加 Product 对象到 `products` 数组。产品卡片和详情页会自动更新。

### 7.5 常见问题

**构建报错 `PageNotFoundError: /_document`**
```bash
rm -rf .next node_modules/.cache && npm run build
```

**端口被占用**
```bash
npx kill-port 3002
```
